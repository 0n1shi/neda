# NES Assembly

## Register

|notation|name|
|-|-|
|A|accumulator|
|X|index register x|
|Y|index register y|
|S|stack pointer|
|P|status register|
|PC|program counter|

## Addressing

|mode|assembler format|
|-|-|
|Immediate|#aa|
|Absolute|aaaa|
|Zero Page|aa|
|Implied||
|Indirect Absolute|(aaaa)|
|Absolute Indexed,X|aaaa,X|
|Absolute Indexed,Y|aaaa,Y|
|Zero Page Indexed,X|aa,X|
|Zero Page Indexed,Y|aa,Y|
|Indexed Indirect|(aa,X)|
|Indirect Indexed|(aa),Y|
|Relative|aaaa|
|Accumulator|A|
